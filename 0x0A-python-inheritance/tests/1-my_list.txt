This file test if the MyList class can sort a list that it inherits

::

        >>> MyList = __import__('1-my_list').MyList
        >>> dlist = MyList()
	>>> dlist += [1, 2, -2, 0, 15]
        >>> dlist.print_sorted()
        [-2, 0, 1, 2, 15]
        >>> print(dlist)
        [1, 2, -2, 0, 15]
        >>> dlist = MyList()
	>>> dlist += [-1, -33, -77, -6, -25]
        >>> dlist.print_sorted()
        [-77, -33, -25, -6, -1]
        >>> print(dlist)
        [-1, -33, -77, -6, -25]
        >>> dlist = MyList()
	>>> dlist += [1.2, 2.3, 7.7, 0.3, 15.2]
        >>> dlist.print_sorted()
        [0.3, 1.2, 2.3, 7.7, 15.2]
        >>> print(dlist)
        [1.2, 2.3, 7.7, 0.3, 15.2]
	>>> dlist = MyList()
	>>> dlist += [1, 3, 1, 2, 3]
	>>> dlist.print_sorted()
	[1, 1, 2, 3, 3]
	>>> dlist = MyList()
	>>> dlist.print_sorted()
	[]
	>>> dlist = MyList()
	>>> dlist += ["bat", "cat", "fat", "app", "got"]
	>>> dlist.print_sorted()
	['app', 'bat', 'cat', 'fat', 'got']
